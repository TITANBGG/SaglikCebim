# Contributing to SağlıkCebim

First off, thank you for considering contributing to SağlıkCebim! It's people like you that make it such a great tool.

## How to Contribute

### 1. Report a Bug
If you find a bug, please use the Bug Report template to report it. Ensure you provide as much detail as possible, including steps to reproduce.

### 2. Suggest a Feature
If you have an idea for a new feature, use the Feature Request template. Describe your idea and how it would benefit the project.

### 3. Submit a Pull Request
- Fork the repository and create your branch from `main`.
- If you've added code that should be tested, add tests.
- Ensure the test suite passes.
- Follow the existing code style (use `flake8` for Python, `eslint` for frontend).
- Issue that pull request!

## Code Style
- **Backend:** Python 3.11+, follow PEP 8.
- **Frontend:** React + TypeScript, follow the ESLint rules defined in the repository.

## Commit Messages
- Use present tense ("Add feature" not "Added feature")
- Use imperative mood ("Move cursor to..." not "Moves cursor to...")
- Limit the first line to 72 characters or less
- Reference issues and pull requests liberally after the first line

Thank you for contributing!
