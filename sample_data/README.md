# Sample Data Directory

This directory is intended for **local testing only**.

To keep the repository clean, lightweight, and privacy-compliant:
- Do **not** commit real patient data, massive datasets, or sensitive medical records here.
- Place a few dummy/test files (e.g., sample X-ray images, placeholder lab PDFs) here to test the `SağlıkCebim` ML and NLP pipelines locally.

The main dataset for training/evaluation is hosted externally (see the main repository README for links).
